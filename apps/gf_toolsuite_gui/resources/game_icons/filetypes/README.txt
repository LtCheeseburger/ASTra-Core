Custom filetype icons for ASTra

Place PNG/SVG/ICO/JPG/WEBP files in this folder using the uppercase type name.
Examples:
- AST.png
- RSF.png
- P3R.png
- DDS.png
- APT.png
- CONST.png
- XML.png
- DB.png
- ZLIB.png
- FOLDER.png

At runtime ASTra looks in:
- <app>/game_icons/filetypes/
- repo/dev copies of apps/gf_toolsuite_gui/resources/game_icons/filetypes/

If no custom icon exists, ASTra falls back to a Qt standard icon.
